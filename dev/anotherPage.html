Detailed API

GrapheneQFT.GrapheneCoord — Type

Lattice coordinate of a carbon atom, generated using graphene_A or graphene_B. Each coordinate contains the sublattice index, as well as the integer coefficients of the two basis vectors $d\times(\pm 1 \hat{x} + \sqrt{3}\hat{y}) / 2$, where $d = 2.46$Å is the graphene lattice constant.

GrapheneQFT.graphene_A — Method
graphene_A(u::Int, v::Int)

Create a GrapheneCoord for an atom belonging to sublattice A at the unit cell (u, v)

Arguments

  • u: coefficient of basis vector $d\times(1 \hat{x} + \sqrt{3}\hat{y}) / 2$
  • v: coefficient of basis vector $d\times(-1 \hat{x} + \sqrt{3}\hat{y}) / 2$

Output

  • GrapheneCoord of the carbon atom
